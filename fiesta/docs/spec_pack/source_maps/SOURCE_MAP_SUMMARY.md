# Source Map Summary

## Object module evidence

The Zone PDB alone exposes hundreds of original object modules. The most important missed systems in the first requirements pack are: AuctionSystem, CardCollection, Gamble/SlotMachine, GuildAcademy, GuildTournament, GuildWar, HolyPromise, InstanceDungeon, KingdomQuest, Minimon/Pet/MiniHouse/Mover, Party/Raid/PartyFinder, PineScript/Lua, QuestFramework, ShineItem* split modules, StreetBooth, TradeBoard, SpamerPenalty, Prison/Admin/OPTool, and full DataReader/Checksum/CommonData layer.

## Current gap statement

The old pack covered a skeleton of protocol/data/character/combat/abstate/item/mob/instance/quest/db/server/GM/Gamebryo. It did not yet cover the full feature families present in the data and PDBs. This rewrite pack expands those families and includes PDB names for reuse.
